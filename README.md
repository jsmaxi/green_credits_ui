# EV Carbon Credits

UI code repository for https://github.com/enderNakamoto/green_credits

Web application https://green-credits.vercel.app/

To get started:

```bash
cd nextjs

npm install

npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
